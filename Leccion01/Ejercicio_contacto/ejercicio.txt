Crear una nueva tabla en tu base de datos:
CREATE TABLE contacto (
    id_contacto SERIAL PRIMARY KEY,
    nombre VARCHAR(50),
    telefono VARCHAR(20),
    email VARCHAR(100)
);

Objetivo del ejercicio:

Crear un programa en Python que permita al usuario:

--Insertar uno o varios contactos.

--Actualizar uno o varios contactos.

--Eliminar uno o varios contactos.

--Mostrar todos los contactos.

--Salir del programa.

Indicaciones:

-Cada operación debe usar transacciones (commit y rollback).

-Usar listas de tuplas para inserciones o actualizaciones múltiples con executemany().

-Validar los datos del usuario (por ejemplo, que id_contacto sea un número entero).

-Mostrar cuántos registros fueron afectados después de cada operación.
